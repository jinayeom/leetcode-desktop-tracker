# LeetCode Tracker
Being a student can be difficult. Endless projects to complete, jobs to hunt for, and courses to pass. 
In the craze of it all, I get that it can be hard to stay disciplined with the LeetCode grind. 

To help with that, I've created a desktop LeetCode tracker so that you can see your progress (or lack of progress) first thing when you open your laptop.

Time to hold yourself accountable and grind that LeetCode!

## Features 
✅ Heatmap layout of progress <br>
✅ Customize heatmap color <br>
✅ Visualize LeetCode progress for the whole year <br>
